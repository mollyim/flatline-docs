# Contributing to Flatline Documentation

Thank you for your interest in contributing to the Flatline project documentation.

## How to Participate

### Working Drafts

Working drafts are documents under active community review in the [`drafts/`](drafts/) directory.

- Use **issues** for questions, missing context, and proposals that need discussion.
- Use **pull requests** for concrete edits to the draft, including `> ...` inline comment blocks.
- Join the **Matrix room** [#mollyim-flatline:matrix.org](https://matrix.to/#/#mollyim-flatline:matrix.org) for informal discussion.
